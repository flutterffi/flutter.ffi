# 翻译即重读

译者在两个语言之间搭桥，损耗不可避免。读《小王子》中英对照，会发现「驯养」与 tame 的温度差。

技术文档翻译同理：deadline 不能直译成「死线」；应说「截止日期」。好的本地化是**二次写作**，不是词典替换。
