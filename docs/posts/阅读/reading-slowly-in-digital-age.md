# 慢阅读

推送把阅读切成碎片，我们误把「看过标题」当作「读过书」。

我尝试每周留出两个**无屏幕小时**：只带一本纸书与一支铅笔。划线不是装饰，而是与作者对话的脚印。

慢不等于少读，而是给大脑留白，让概念之间发生连接——这在扫描 API 文档时同样适用：深读一章胜过收藏十篇。
