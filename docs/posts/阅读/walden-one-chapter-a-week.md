# 每周一章瓦尔登

梭罗若活在今日，大概会嘲笑我们买齐机械键盘却写不出三百字日记。

「简朴」不是苦行，而是**看清真正需要的工具**。我的编辑器插件删到只剩语法高亮与 Git，像给思想清障。

湖在麻省，也在每次关机后的安静里。
