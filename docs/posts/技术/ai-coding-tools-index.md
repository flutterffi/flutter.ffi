## 选型一句话

- **不想换编辑器** → Copilot 补全，或 Cline / Roo Code 二选一  
- **要 AI-native IDE** → Cursor、Windsurf、Trae 各试用后定主力  
- **要终端自动化** → Claude Code、Codex 或 Gemini CLI  
- **要架构/调试分工** → Roo Code 的 Architect / Debug 模式  
- **要换模型、控成本** → Cline / Roo Code + 自有 API，或 GLM 等套餐（见收藏区）
