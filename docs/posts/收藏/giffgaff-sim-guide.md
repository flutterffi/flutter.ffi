# 经典 Giffgaff 开卡指南（英国平价 SIM）

[Giffgaff](https://www.giffgaff.com) 是英国常见的虚拟运营商（MVNO），走 O2 网络，以**无合约、套餐灵活、可免费领卡**著称。适合留学、短期旅行或需要英国号码接收短信的场景。本文作收藏备用，具体资费以官网为准。

## 为什么常被收藏

- 可**免费申请实体 SIM** 邮寄（英国地址；部分地区也支持 eSIM）
- 无长期合约，按月/按流量购买
- 社区论坛客服响应快，教程多
- Pay as you go，不用不扣高额月费

## 开卡前准备

| 项目 | 说明 |
|------|------|
| 邮箱 | 注册与收激活邮件 |
| 英国地址 | 实体卡邮寄；可请朋友代收或宿舍地址 |
| 解锁手机 | 确保手机支持英国频段（一般欧版/国际版即可） |
| 支付方式 | 借记卡 / PayPal 等（激活套餐时） |

## 第一步：申请免费 SIM

1. 打开 [giffgaff.com/order-free-sim](https://www.giffgaff.com/order-free-sim)  
2. 选择 **Free SIM**（或 eSIM，视设备与地区）  
3. 填写邮寄信息，提交订单  
4. 通常数个工作日收到卡（繁忙季可能更久）

> 通过好友推荐链接申请，双方往往有额外话费奖励；可自行搜索「giffgaff referral」选用。

## 第二步：激活 SIM

收到卡后：

1. 插入手机，开机  
2. 浏览器访问卡上印刷的激活地址，或登录官网 **Activate your SIM**  
3. 输入 SIM 卡上的 **6 位激活码**（Activation code）  
4. 选择首个 **Goodybag**（月套餐）或 **Credit only**（仅充值、按量扣费）  
5. 绑定支付方式并完成首充  

激活成功后会有确认短信，号码即英国手机号（+44）。

## 套餐怎么选（思路）

Giffgaff 套餐叫 **Goodybag**，常见类型：

- **流量型**：大流量 + 无限通话短信（适合日常主卡）  
- **少量流量**：便宜，适合只收验证码  
- **Always on**：低价保号向，流量很少  

若只是**收短信、偶尔上网**，可先选低档 Goodybag 或只充值 credit，按需购买。

在官网 **Top up / Buy goodybag** 可随时更换下月套餐；当月套餐快用完也可提前续买（注意规则说明）。

## 第三步：保号与长期使用

Giffgaff 为预付费，长期不用可能因余额/活动规则被回收号码。收藏本指南时请记住：

- 定期登录 App 或官网查看余额与套餐到期日  
- 需要保号时，可购买最便宜的 goodybag 或按官方说明维持活跃  
- 离开英国前评估是否还需要该号，避免自动续费  

具体保号政策以 Giffgaff 当前条款为准。

## 常用操作速查

```text
查余额 / 套餐：官网登录 或 手机快捷码（见官网 Help）
充值：Top up → 选金额或 Goodybag
换套餐：下月生效的 goodybag 可在账户里更换
丢失 SIM：在官网申请 SIM 替换（可能收费）
```

英国本地拨打短号、国际漫游等，请在 **Help → Roaming** 查看目标国家资费。

## 常见问题

### 在国内能提前激活吗？

实体卡通常需在英国或支持邮寄的地区完成激活；eSIM 规则不同，以官网说明为准。一般建议**落地英国后再激活**，避免套餐计时过早开始。

### 支持热点吗？

多数 Goodybag 含 tethering，购买前在套餐详情页确认 **tethering included**。

### 能转网带走号码吗？

支持 PAC 码转网（Porting），在账户里申请 PAC，提供给新运营商即可。
