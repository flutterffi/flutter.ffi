# 欢迎 👋

这是你的 Flutter FFI 技术博客的第一篇文章！

## 博客特点

- 🚀 **自动化部署** - 推送到 GitHub 自动构建部署
- 📝 **Markdown 写作** - 简单高效的 Markdown 编辑
- 🎨 **Fluid 主题** - 优雅简洁的界面设计
- 🔍 **SEO 友好** - 自动生成 sitemap

## 开始写作

```bash
# 创建新文章
hexo new "我的第一篇文章"

# 本地预览
hexo server

# 部署到 GitHub Pages
git add .
git commit -m "添加新文章"
git push origin main
```

## 目录结构

```
blog-source/
├── source/
│   └── _posts/          # 博客文章
├── themes/              # 主题
├── .github/
│   └── workflows/       # CI/CD 配置
└── _config.yml          # 博客配置
```
